Routes and the handlers associated with them can be defined in app.js. This could lead to a very large file. A router can be created as a module. This can have middleware as well as some routes. A Code Example is given here of a router with one route in it and no middleware.

Code Example: different-route

// routes/different-route.js

const express = require('express');
const router = express.Router();

router.get('/', function (req, res) {
	res.send('this is a different route!!');
});

module.exports = router;

This shows a minimal implementation of a route using the express.Router class.
